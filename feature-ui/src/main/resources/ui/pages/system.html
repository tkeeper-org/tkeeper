Promote to threshold

Move this keeper from mono custody into an MPC quorum.
Irreversible operation
TKeeper will distribute existing keys to the target peers as threshold shares, then delete mono key material and previous mono generations. This keeper cannot be moved back to mono mode after promotion.
Target peers must already be initialized, unsealed, reachable, and configured for the same quorum.
